## 今天真正掌握了什么 (L4)

- **JSON 是合法字符串的一个子集**: 必须 `""` 包字符串、`{}` 包对象、`[]` 包数组;裸词不能解析 (实验 1+3 验证)
- **正则 `(...)` 是捕获组**: `match.group(N)` 拿第 N 个捕获的内容 (LLM 回复里 [TOOL:print:{"text":"hi"}] -> group(1)="print", group(2)='{"text":"hi"}')
- **正则 vs json.loads 职责分工**: 正则只切不解析,json.loads 负责"读懂"JSON
- **`return response` 是函数当场退出**: 不是"继续循环"
- **`if not match` 利用了 None 是 falsy**: 搜不到时 search() 返回 None
- **真实 DeepSeek API 请求/响应**端到端走通
- **Agent Loop 数据轨迹**: str -> Match -> str -> dict -> Tool Object -> function -> str
