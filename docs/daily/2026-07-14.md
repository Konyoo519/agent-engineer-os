## 笔记：`TOOLS[tool_name]["func"]()` 拆解

| 部分              | 含义                              |
| ----------------- | ------------------------------- |
| `TOOLS`          | 整个工具注册表（dict）                  |
| `[tool_name]`    | 用名字动态选工具                       |
| `["func"]`       | 这个工具的 key（指向函数对象）             |
| `()`             | 真正执行函数                         |

**需求变更时改哪里：**

- 新增工具 → 在 TOOLS 加一项
- 改工具逻辑 → 改对应函数
- 工具元数据 → 改对应 dict 项

**报错定位思路：**

- `KeyError: 'xxx'` → TOOLS 里没有这个名字
- `TypeError` → 工具签名和调用方式不匹配
- 函数自身异常 → 看 `func` 指向的函数体
